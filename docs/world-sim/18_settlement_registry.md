# Settlement Registry

## Purpose

Settlement registry — это канонический список поселений карты.

Он нужен для:
- будущей торговли;
- будущей мультигородской симуляции;
- связи карты мира с экономикой;
- избежания случайных тестовых поселений в основной модели.

## Map coordinates

- Координаты нормализованы относительно базового изображения карты (`X` и `Y` в диапазоне `0..1`).
- Координаты используются для map markers, будущих торговых маршрутов и выбора поселений на карте.
- Эти координаты не являются географическими широтой/долготой.
- Координаты должны оставаться стабильными, пока не меняется базовое изображение карты.

## Settlement economy profile

Каждое поселение должно иметь `SettlementEconomyProfile`, который определяет доступные источники экономики (земледелие, рыбалка, охота, поставки с материка, ресурсы и ремесло) и роль поселения (порт/крепость/столица).

## Implementation rule

Правила:
- Все settlement presets должны создаваться только из settlement registry.
- Тестовые поселения можно использовать только в тестах или dev-документах.
- Основная модель мира не должна содержать выдуманные settlement names.
- Id должны быть стабильными и ASCII-compatible.
- DisplayName может содержать кириллицу и диакритику.
