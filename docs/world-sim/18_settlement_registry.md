# Settlement Registry

## Purpose

Settlement registry — это канонический список поселений карты.

Он нужен для:
- будущей торговли;
- будущей мультигородской симуляции;
- связи карты мира с экономикой;
- избежания случайных тестовых поселений в основной модели.

## Implementation rule

Правила:
- Все settlement presets должны создаваться только из settlement registry.
- Тестовые поселения можно использовать только в тестах или dev-документах.
- Основная модель мира не должна содержать выдуманные settlement names.
- Id должны быть стабильными и ASCII-compatible.
- DisplayName может содержать кириллицу и диакритику.
