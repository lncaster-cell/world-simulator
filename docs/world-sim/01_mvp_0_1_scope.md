# 01 — MVP 0.1 Scope

## In scope

- One city only: Gotha (`city_gotha`).
- Time progression: Start/Pause, hourly ticks, daily recalculation.
- City parameters and city_state model.
- Resource flow model for food/economy baseline.
- Event model (concrete incidents, not parameter labels).
- Main map screen + city card + city panel.
- Save/Load model.
- Game log + technical log.
- Debug panel for balancing.

## Strict scope lock

MVP 0.1 must stay minimal and must not be expanded without a separate planning/approval step.
