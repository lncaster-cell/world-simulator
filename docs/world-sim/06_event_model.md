# 06 — Event Model

## Definition

An event is a concrete incident, not a renamed parameter/debuff.

Examples:

- artists performance;
- market fair;
- warehouse fire;
- resident disease outbreak;
- rat infestation;
- major theft;
- patrol reinforcement;
- poor harvest;
- good harvest;
- city rumors.

## Event schema (required)

Each event must have:

- `id`;
- title;
- type;
- cause;
- spawn conditions;
- duration;
- effect;
- cooldown;
- status;
- log record.
