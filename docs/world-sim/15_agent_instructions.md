# 15 — Agent Instructions for MVP Discipline

## Non-expansion policy

- Do not expand MVP without explicit separate approval.
- Keep focus on one city (`city_gotha`).
- Do not introduce application code during documentation-initialization tasks.

## Forbidden additions at this stage

- Unity setup;
- NWN2 runtime integration;
- multi-city systems;
- caravans/armies/clans/states/wars.

## Working order

1. Documentation first.
2. Review/approve documentation.
3. Plan implementation tasks later.
