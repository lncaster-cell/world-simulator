# 00 — Overview

## Purpose

Define MVP 0.1 documentation baseline for an external Windows world simulator used by an NWN2 module.

## Product boundaries

- Separate Windows application (not Unity).
- Simulator core runs independently from NWN2.
- Single-city scope for MVP: Gotha.
- No app code implementation in this phase; docs only.

## Core principle

Do not expand MVP without a separate explicit decision.
