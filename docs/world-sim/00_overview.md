# 00 — Overview

## Purpose

Define MVP 0.1 documentation baseline for an external Windows world simulator used by an NWN2 module.

## Product boundaries

- Separate Windows application (not Unity).
- Simulator core runs independently from NWN2.
- Single-city scope for MVP: Gotha.
- Historical note: this document was created during the original docs-only phase; the repository now contains a working application.

## Core principle

MVP boundaries in this document are historical planning context, not an active prohibition. Agents may expand or revise them when the user's request and current project state require it, while keeping documentation and tests aligned.
