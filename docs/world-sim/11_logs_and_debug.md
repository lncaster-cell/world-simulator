# 11 — Logs and Debug

## Log types

1. Game journal — readable chronology.
2. Technical log — explicit calculation breakdown.

## Technical log example

Day 5:

- Food: 1000 → 910
- population consumption: -84
- fishing: TBD
- hunting: TBD
- mainland supplies: TBD
- event “rats”: TBD
- total: -90

## Debug panel (required)

Must allow:

- add/remove food;
- add/remove wealth;
- add/remove resources;
- add/remove goods;
- set mood;
- set security;
- set crime;
- trigger event manually;
- skip hour;
- skip day;
- save;
- load;
- reset Gotha to start preset.

## Technical log retention

- Technical log shows only the latest 500 entries.
- Older entries are removed automatically from the UI log.
