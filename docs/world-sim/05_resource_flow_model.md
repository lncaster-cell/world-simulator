# 05 — Resource Flow Model

## Food flow

- Food measured in storage units.
- Consumption: `population * 0.2` per day.

For Gotha at 420 population:

- daily consumption: 84 food.

## Daily food change components

- Base consumption (negative).
- Fishing output.
- Hunting output.
- Mainland supply deliveries.
- Event modifiers (e.g., rats, disease, warehouse fire).

- Numeric production formulas can start with TBD placeholders in technical logs and later be finalized during balancing.
